# Installation guide

- create a database
- `cp .en.example .env`
- `composer install`
- `php artisan migrate`

Start the interactive Q and A using this command :

- `php artisan qanda:interactive`

Reset the progress using this command :

- `php artisan qanda:reset`

Run the unit tests using this command :

- `php artisan test`
